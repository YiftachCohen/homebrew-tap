# homebrew-tap

Homebrew tap for [Token Tab](https://github.com/YiftachCohen/token-tab) — a
provably-safe Claude Code usage meter for the macOS menu bar.

```sh
brew tap yiftachcohen/tap
brew install --cask token-tab
```

The cask installs the same Developer ID-signed, notarized `Token Tab.app` published on
the [releases page](https://github.com/YiftachCohen/token-tab/releases), verified
against its pinned sha256.
